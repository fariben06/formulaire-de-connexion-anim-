# Formulaire de connexion animé

## Formulaire de connexion animé

- Formulaire de connexion animé utilisant HTML CSS et JavaScript
- Il contient un arrière-plan élégant et magnifique.
- Avec animation d'étiquettes flottantes.
- Et un bouton pour afficher et masquer le mot de passe.
- Développé d'abord avec la méthodologie Mobile First, puis pour le bureau.
- Compatible avec tous les appareils mobiles.
